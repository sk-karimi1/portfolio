About me

Information about me

I am currently studying BSc Computer science at the University of Sheffield. I am a first year student and I am enjoying my time here. My graduation year will hopefully be in 2025.

Life at University

University has been a very new experience for me. It is the first time that I have been living by myself, which also comes with new challenges. Its very nice to meet a lot of new people and get to know them all. I have been fortunate enough to have been in contact with other university students prior to when I started, where I asked them what I should expect coming into University, which helped me alot.

My Modules

I am currently doing 6 modules at the moment and I will discuss briefly about what they are.

COM1001 Introduction to Software engineering:

This module goes through Ruby On Rails, which is a programming language uses for computers, we are currently going through the basics of how this language works, and we will go through more complex ideas later on in the year. We are also learning about an operating system called Linux, where we have learnt commands to create files, edit user permissions, editing files, etc by just using the terminal window. Linux is very interesting as it is a brand new concept for me so I enjoy this alot.

Our lectures on this module have been about software and the main methods of how it is created for clients and companies. We have learnt concepts such as Agile and Waterfall, both of which are methods to create software, each having its own advantages and disadvantages. We also have an assignment where we plan on creating software for a client and we will be making it next semester.

COM1002 Foundation of Computer Science:

I would say that this module is one of the more difficult subjects compared to the others. This module goes through logic, such as propositional and predicate logic, which is fundamental in computer science.


This is an example of what predicate logic looks like above. It looks quite complex to someone who hasnt seen it before (it still looks complex for me), but it is quite fun once you understand it.

COM1003 Java Programming:

This module is quite self-explanatory. We learn the Java programming language and implement it, by doing exercises and assignments. We have leanrt key fundamental concepts for programming in java, such as conditional loops, arrays, strings etc. I am still quite new to this language and will hopefully leant more in the future.

There are hundreds of different programming languages. The more famous ones being Ruby (which I had mentioned before), Java, Python, Javascript, C++, C#, HTML and CSS for website building and much more. I plan on learning more languages so I can strengthen my knowledge on programming and become more experienced

COM1005 Machines and Intelligence:

Quite an interesting module this is. We go through the history of AI and how it has developed and changed over time. Out lab class goes through MiroCode, which is a robot dog where we can control its movement and its behaviour using code.

COM1006 Devices and Networks:

This module focuses on topics such as binary, logic gates, circuits etc. Its main foucus is to understand how simple devices work. It may seem a bit complicated at first, however it is quite straight forward and one of the easier modules.
